Task 4

Since organization is essential for any data science project or work, we should begin by setting up an organizational structure for this course and its contents.

  1. Setup a directory/folder on your laptop that will contain all your materials and work for this course. You may already have one setup and named. We will setup this directory and call it stat301-1. Where this is located on your laptop is completely up to you.

  2. Within stat301-1/, place a copy of the course syllabus for future reference. We also will create a new sub-directory that will contain all labs for this course. For ease let us call it labs. So now we have stat301-1/labs/.

  3. Download L01_introduction.zip from Canvas.

  4. Unzip/decompress L01_introduction.zip and place the unzipped/decompressed folder in /stat301-1/labs. So you should now have stat301-1/labs/L01_introduction/. The L01_introduction/ sub-directory contains all materials for lab L01 Introduction.

  5. Open RStudio and create a project for the existing directory L01_introduction/.

  6. Rename template_L01.qmd to your Lastname_Firstname_L01.qmd

  7. Render the file with Cmd/Ctrl + Shift + K or click the Render button.
